# Demo


## Run Locally

Clone the project

```bash
  git clone https://github.com/Zeeshan2125/EX-Bouquet.git
```

Go to the project directory

```bash
  cd flower-for-thought-ecommerce-react
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
